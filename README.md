#QUnit Logging

Logging your [QUnit](http://qunitjs.com/) tests like a pro!

##Usage

Include in your html:
```html
<script type="text/javascript" src="qunit.js"></script>
<script type="text/javascript" src="qunit-logging.js"></script>
```
